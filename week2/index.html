Notes

  • Programming is about making computers do what you want. There are three different programming languages: low level programming languages which consist of machine code and assembly language and are closely associated with a computer's hardware.
  • High level programming languages consists of codes that are easier for humans to read and write. Such examples are C, C++ or Java, which is then compiled into machine code and executed. Due to the level of these programming languages, it makes the programs very fast.
  • Scripting languages are also high-level, but they are interpreted, which means they are translated into machine code at run time. JavaScript is one of the most popular programming languages in the world and nearly all web browsers can run it. JavaScript is a high-level scripting language that is interpreted and compiled at run time.
  • A well-commented code is very important for a programmer. It makes it easier for anybody reading the code to understand what's going on, and that includes the programmer.

JavaScript Grammar

Statements:

Block:

A block is a series of statements that are collected together inside curly braces

Blocks do not need to be terminated by a semicolon.

Variables

From ES6 onwards, JavaScript uses the keywords "const" and "let" to declare variables.

They are block scoped, so their value only exists inside the block they are declared in.

Naming Constants & Variables

When naming constants and variables, it is important to give them sensible names that describe what the variable represents

The folowing are examples of naming variables:

$name

_answer

firstName

last_name

address_line1

Template Literals

Template literals use the backtick character, `, to deliminate the string.

const name = `Siri`;

`Hello ${ name }!`;

'Hello Siri!'

const age = 39;

`I will be ${ age + 1 } next year`;

'I will be 40 next year'

Symbols

const uniqueID = Symbol('this is a unique ID');

It is possible for two variables to point to the same symbol if the for() method is used when the symbol is created:

const A = Symbol.for('shared symbol');

const B = Symbol.for('shared symbol');

Arrays, Logic, and Loops

Arrays

Array is an ordered list of values.

const myArray = [];

const myArray = new Array();

Adding Values to Arrays

heroes[0] = "Superman";

Removing Values from Arrays

delete avengers[3];

Destructuring Arrays

const [x,y] = [1,2];

Array Properties and Methods

const avengers = ['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow'];

avengers.length;

avengers[avengers.length - 1];

Unshift, Push, Shift, Pop

avengers.unshift('Captain America'); /Appends a new value to the begin

avengers.push("Thor"); /Appends a new value to the end

avengers.shift("Thor"); /To remove the first item

avengers.pop(); /To remove the last item

Merging Arrays

avengers.concat(['Hulk','Hawkeye', 'Black Widow']);

The Join Method

The join() method can be used to turn the array into a string that comprises all the items in the array, separated by commas:

avengers.concat(['Hulk','Hawkeye', 'Black Widow']);

avengers.join();

'Captain America, Iron Man, Thor, Hulk, Hawkeye, Black Widow'

Slicing and Splicing

avengers.slice(2,4) // starts at the third item (index of 2) and finishes at the fourth (the item with index 4 is not included)

avengers.splice(3, 1, 'Scarlet Witch'); //removes items from an array then inserts new items in their place

Sets

A set is a data structure that represents a collection of unique values, so it cannot include any duplicate values.

Functions

A function is a chunk of code that can be referenced by a name, and is almost like a small, self-contained mini program.

Function Declarations

function hello(){

console.log('Hello World!');

}

A function declaration ends in a block (there's no need for semicolons at the end of blocks).

Function Expressions

const goodbye = function(){

console.log('Goodbye World!');

};

const goodbye = function bye(){

console.log('Goodbye World!');

};
